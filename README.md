# Classification-of-breast-cancer-subtypes-using-random-forest-
Analyzed breast cancer RNA-seq data with PAM50 subtypes (LumA, LumB, HER2, Basal, Normal). Processed VST-normalized counts, selected 2,000 high-variance genes, Perfromed baseline random forest with n estimators 500 ,random forest with SMOTE and SMOTE+TOMEK ,Model evaluation is done with confusion matrix and classification report
